# グループキュー

グループキューは複数のキューをまとめ、様々な実行方法で実行することができるキューです。

## グループキューのエディター項目

![グループキュータブ](group_tab.png)

- **グループ実行モード**

    グループ内のキューの実行方法を設定できます。
    以下の実行方法から選択できます。

    - **プレイリスト**
        先頭のキューから順番に実行します。
        １つのキューが完了するごとに次のキューが実行されます。

    - **同時実行**
        含まれるすべてのキューを同時に実行します。

- **リピート**

    **グループ実行モードが** が **プレイリスト** モードであるときに設定できます。

    グループ内の最後のキューが完了したとき、先頭のキューを実行します。

    これにより、グループ内のキューを順番に繰り返し再生することができます。
