# 再生制御キュー

対象のキューに対して特定の再生制御コマンドを適用することができます。
効果としては、対象のキューを選択してそれぞれのショートカットキーを押した場合と変わりません。

種類としては以下のものがあります。

- **開始キュー**
- **停止キュー**
- **一時停止キュー**
- **読み込みキュー**

また、再生制御キューの特徴として、実行のための所要時間を持たないため、実行中のキューには表示されません。

## 再生制御キューのエディタ項目

![プレイバックキュータブ](playback_tab.png)

- **対象キュー**

    再生制御を行う対象のキューです。
    自身以外のキューを指定できます。
